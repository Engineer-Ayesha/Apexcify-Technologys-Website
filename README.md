# Apexcify Technologies Website

A modern and responsive company website built using HTML, CSS, and JavaScript. This project showcases a clean UI design with interactive elements and smooth user experience.

## 🚀 Features

* Responsive design for all screen sizes
* Modern and clean user interface
* Smooth scrolling navigation
* Interactive portfolio section
* Contact section with direct links (Email, LinkedIn, WhatsApp)
* Hover effects and transitions

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
